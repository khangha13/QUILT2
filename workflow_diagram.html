1 Panel Preparation

Normalises the reference panel by standardising chromosome naming conventions (e.g. Chr01 → chr1) and/or filtering out sites with low phased-genotype rates. Runs as a SLURM array per chromosome.

bcftools annotate bcftools view tabix --standardise-name --remove-missing --min-valid-gt-rate
2 Chunk Definition

Divides each chromosome into genomic chunks for parallel processing. Supports automatic chunking via QUILT's R function, a user-supplied chunk TSV, or explicit start/end region boundaries.

QUILT::quilt_chunk_map --auto-chunk-map --chunk-file --region-start/end
3 Reference Preparation

First half of the SLURM array job (per chunk). Builds a QUILT2 reference object (.RData) from the phased panel VCF, genetic map, and region boundaries. Skipped if --impute-only and cached .RData exists.

QUILT2_prepare_reference.R --nGen --buffer --outputdir
4 Imputation

Second half of the SLURM array job (per chunk). Imputes diploid genotypes for all low-pass samples using the pre-built .RData reference and the sample BAM list. Per-chunk VCFs are then concatenated with bcftools concat.

QUILT2.R bcftools concat --bamlist --nGen --impute-only
5 Evaluation

Post-imputation accuracy assessment comparing imputed genotypes against a high-depth truth VCF. Derives dosage from GT fields, removes strand-ambiguous loci, and reports per-variant r², concordance, and per-sample/MAF-bin summaries.

dosage_r2.R vcfppR bcftools data.table ggplot2 --truth-vcf
